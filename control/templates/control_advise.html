Roboterzustand
HINWEIS
Es wurde kein $control_advise - Eintrag durchgeführt. Bitte erneut auf den Roboterzustand klicken und Eintrag durchführen. Wahrscheinlich wurde ein Pflichtfeld ausgelassen.

Dieser Hinweis tritt dann auf, wenn in der Datenbank kein Eintrag für einen Host und/oder für eine Komponente gefunden wurde. Es wird vorausgesetzt, dass beim ersten Start dieses Plugins zunächst ein Host eingerichtet wird und im Anschluss darauf eine Komponente. Pflichtfelder müssen ausgefüllt werden!

Zusatzbemerkung: Der Name einer Komponente darf nicht "Neu" heissen!

Solte dieser Fehler erneut auftauchen, so können folgende Hinweise nützlich sein:
  • Sind die Zugriffrechte für die Datenbank richtig?
  • Sind in der Datenbank die Tabellen "robot_hosts" und "robot_components" vorhanden, bzw. werden diese Automatisch angelegt?
